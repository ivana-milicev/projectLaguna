Browsers Tested: Chrome, Firefox, MS Edge  
Test Framework: Selenium (Java) & Playwright (JavaScript)


* TEST SCOPE

Smoke testing covers critical user flows to verify that the main functionalities of the Laguna bookstore website work correctly.
These tests ensure that the core features of the e-commerce platform are operational before proceeding with more detailed testing.

Areas Covered:
- Search functionality
- Login functionality
- Complete buy flow (payment excluded)
- Checkout process(payment excluded)
